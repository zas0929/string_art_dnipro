# String Art Generator

Статический прототип генератора string art.

## Как работает

1. Загружает исходное изображение.
2. Позволяет кадрировать фото зумом и перетаскиванием. Предпросмотр показывает оригинальный кадр, а для расчета нитей строится внутренняя тональная карта с мягким усилением контуров.
3. Расставляет пронумерованные точки по окружности.
4. Жадно выбирает следующую линию нити: каждая новая линия должна проходить через самые темные еще не покрытые области.
5. Выдает предпросмотр и инструкцию сборки: последовательность соединения точек.

## Запуск

Откройте `index.html` в браузере.

Для результата ближе к мобильному референсу используйте `300` точек и `4500` линий. Если нужно быстро проверить механику, временно ставьте `180-240` точек и `800-2000` линий.

По умолчанию используется `240` точек. После загрузки фото его можно двигать мышью/тачем на правом предпросмотре и масштабировать ползунком `Зум фото` или колесом мыши.
