# String Art Generator

Статический прототип генератора string art.

## Как работает

1. Загружает исходное изображение.
2. Позволяет кадрировать фото зумом и перетаскиванием. Предпросмотр показывает оригинальный кадр, а для расчета нитей строится внутренняя тональная карта с мягким усилением контуров.
3. Расставляет пронумерованные точки по окружности.
4. В стабильном режиме v4 выбирает следующую линию по остаточной оптической плотности.
5. Экспериментальный v5 дополнительно оценивает изображение на масштабах `560`, `280` и `140` пикселей и периодически проверяет пару следующих линий.
6. Выдает предпросмотр и инструкцию сборки: последовательность соединения точек.

Расчет режимов v4/v5 выполняется в Web Worker. Интерфейс остается отзывчивым, прогресс приходит пакетами, а кнопка `Стоп` сразу завершает worker и сохраняет уже построенную часть инструкции.

## Запуск

Из-за модульного расчетного ядра приложение нужно открывать через локальный сервер:

```bash
python3 -m http.server 5174
```

Затем откройте `http://localhost:5174/`.

Для качественного портрета используйте `240` точек и `4500` линий. Если нужно быстро проверить механику, временно ставьте `180-240` точек и `800-2000` линий.

По умолчанию используется `240` точек и размер картины `47` см. После загрузки фото его можно двигать мышью/тачем на правом предпросмотре и масштабировать ползунком `Зум фото` или колесом мыши.

## Сравнение схем

Показатели маршрута двух TXT-схем можно сравнить командой:

```bash
node tools/compare-schemes.mjs reference.txt candidate.txt
```

Скрипт проверяет среднюю длину перехода, распределение точек и направлений, повторные хорды, параллельные линии и немедленные возвраты.
