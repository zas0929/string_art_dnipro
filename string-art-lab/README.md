# String Art Lab

Python-лаборатория для поиска более качественного алгоритма string art.

Цель этой папки - быстро тестировать разные методы подготовки целевой карты и оптимизации линий, не ломая основной веб-прототип.

## Быстрый запуск

```bash
python3 run.py "../2026-07-13 14.55.15.jpg" --profile portrait --points 240 --lines 4500 --size 47 --out outputs/girl-portrait
```

Для быстрой проверки:

```bash
python3 run.py "../2026-07-13 14.55.15.jpg" --profile portrait --points 160 --lines 800 --out outputs/quick-test
```

## Что создается

В папке результата:

- `source_crop.png` - кадрированное исходное фото.
- `target_map.png` - внутренняя тональная карта, по которой оптимизируется нить.
- `preview.png` - итоговый string-art макет.
- `sequence.txt` - инструкция соединения точек.
- `sequence.csv` - таблица `step,from,to`.
- `report.json` - параметры запуска.

## Профили

- `raw` - почти прямой перевод яркости в темноту.
- `portrait` - мягкая тональная карта для портретов: локальный контраст, контуры, защита светлых лиц.
- `high_detail` - более агрессивные контуры и тени.

## Идея алгоритма

1. Фото кадрируется в квадрат.
2. Строится внутренняя черно-белая карта темноты.
3. По окружности расставляются точки.
4. На каждом шаге выбирается линия, которая сильнее всего уменьшает оставшуюся темноту и не слишком перетягивает уже набранные зоны.
5. Экспортируется предпросмотр и инструкция физической сборки.
